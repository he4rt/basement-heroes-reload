# Dashboard

O dashboard é a primeira página que você vê após fazer login. Ele fornece uma visão geral rápida da sua conta e da plataforma.

## Widgets

O dashboard exibe dois widgets:

- **Widget de Conta** — Mostra seu nome e informações da conta em um relance
- **Widget de Informações** — Exibe a versão da plataforma e links úteis

## Navegação

A partir do dashboard, use a barra lateral para navegar para qualquer seção do painel administrativo. Você sempre pode retornar ao dashboard clicando no logotipo do aplicativo ou no link **Dashboard** no topo da barra lateral.
