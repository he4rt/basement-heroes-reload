# Gerenciando Usuários

Navegue até **Gestão de Usuários > Usuários** na barra lateral para acessar a lista de usuários.

## Lista de Usuários

A tabela de usuários exibe todos os usuários registrados com as seguintes colunas:

- **Nome** — O nome de exibição do usuário (pesquisável)
- **E-mail** — O endereço de e-mail do usuário (pesquisável)
- **E-mail Verificado Em** — A data e hora em que o e-mail foi verificado, se aplicável

Você pode ordenar a tabela por qualquer coluna e usar a **barra de busca** para filtrar usuários por nome ou e-mail. Os usuários também podem ser encontrados pela barra de busca global no topo do painel.

## Criando um Usuário

Clique no botão **Novo Usuário** no topo da lista. Preencha os seguintes campos:

- **Nome** — O nome completo do usuário (obrigatório)
- **E-mail** — Um endereço de e-mail único para este usuário (obrigatório)
- **E-mail Verificado Em** — Opcionalmente, defina uma data para marcar o e-mail como já verificado
- **Senha** — Defina uma senha inicial para o usuário (obrigatório na criação, opcional na edição)

Clique em **Criar** para salvar o novo usuário.

## Editando um Usuário

Clique na ação **Editar** em qualquer linha de usuário, ou clique no nome do usuário para abrir a visualização detalhada e editar a partir dela. Atualize os campos necessários e clique em **Salvar**.

Ao editar, o campo de senha é opcional — deixe-o em branco para manter a senha atual.

## Excluindo um Usuário

Use a ação **Excluir** em qualquer linha de usuário ou na página de edição. Será solicitada confirmação antes da remoção.

Você também pode selecionar múltiplos usuários usando as caixas de seleção e usar a ação de **Exclusão em Massa** para removê-los de uma vez.

> **Atenção:** A exclusão de um usuário é permanente e não pode ser desfeita.

## Personificando um Usuário

Administradores podem fazer login temporariamente como outro usuário para solucionar problemas ou verificar sua experiência. Clique na ação **Personificar** em qualquer linha de usuário para mudar para a sessão desse usuário.

Durante a personificação, um banner aparece no topo da página com a opção de **parar de personificar** e retornar à sua própria conta.

> **Nota:** Todas as ações realizadas durante a personificação são registradas na conta do usuário personificado.
