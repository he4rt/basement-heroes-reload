# Configuração

Principais opções de configuração disponíveis na plataforma Sycorax.

## Tema

O painel suporta os modos **claro** e **escuro**. O modo do tema é aplicado automaticamente com base nas preferências do seu sistema. Cada usuário também pode personalizar sua **cor de destaque primária** na página de Perfil — esta é uma configuração pessoal e não afeta outros usuários.

## Localização

A plataforma suporta múltiplos idiomas:

- **Inglês** (en)
- **Português - Brasil** (pt_BR)

Os usuários podem trocar o idioma preferido na página de **Perfil**. Todos os rótulos, grupos de navegação, mensagens do sistema e documentação serão atualizados para o idioma escolhido.

## Cache da Documentação

A base de conhecimento integrada armazena as páginas de documentação em cache para carregamento mais rápido. Por padrão, as páginas são armazenadas em cache por **24 horas**. Se você notar que alterações na documentação não estão aparecendo, entre em contato com o administrador do sistema para limpar o cache.
