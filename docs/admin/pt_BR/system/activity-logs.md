# Logs de Atividade

Os logs de atividade fornecem um registro completo de auditoria de todas as alterações feitas nos registros da plataforma. Navegue até **Sistema > Logs de Atividade** para visualizá-los.

## O Que É Registrado

Cada ação de criação, atualização e exclusão em recursos-chave é automaticamente registrada. Cada entrada de log captura:

- **Causador** — O usuário que realizou a ação
- **Assunto** — Qual registro foi afetado (ex.: um usuário, equipe ou cargo específico)
- **Evento** — O tipo de ação (criado, atualizado, excluído)
- **Alterações** — Uma comparação dos valores antigos e novos dos campos atualizados
- **Data/Hora** — Quando a ação foi realizada

## Navegando pelos Logs

A tabela de logs de atividade permite navegar por todos os eventos registrados. Você pode:

- **Buscar** por entradas específicas
- **Ordenar** por data ou outras colunas
- **Ver detalhes** de qualquer entrada para visualizar a lista completa de alterações

## Por Que os Logs de Atividade São Importantes

Os logs de atividade ajudam você a:

- **Rastrear alterações** — Veja quem alterou o quê e quando
- **Auditar acessos** — Revise quais usuários realizaram ações sensíveis
- **Solucionar problemas** — Rastreie até a alteração exata que causou um problema
- **Atender requisitos de conformidade** — Mantenha um registro de todas as modificações de dados

## Retenção

Os logs de atividade são mantidos indefinidamente por padrão. Entre em contato com o administrador do sistema se precisar configurar uma política de retenção.
