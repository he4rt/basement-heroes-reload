# Managing Users

Navigate to **User Management > Users** in the sidebar to access the users list.

## User List

The users table displays all registered users with the following columns:

- **Name** — The user's display name (searchable)
- **Email** — The user's email address (searchable)
- **Email Verified At** — The date and time the email was verified, if applicable

You can sort the table by any column and use the **search bar** to filter users by name or email. Users are also searchable from the global search bar at the top of the panel.

## Creating a User

Click the **New User** button at the top of the list. Fill in the following fields:

- **Name** — The user's full display name (required)
- **Email** — A unique email address for this user (required)
- **Email Verified At** — Optionally set a date to mark the email as already verified
- **Password** — Set an initial password for the user (required on creation, optional when editing)

Click **Create** to save the new user.

## Editing a User

Click the **Edit** action on any user row, or click the user's name to open the detail view, then edit from there. Update any fields as needed and click **Save**.

When editing, the password field is optional — leave it blank to keep the current password.

## Deleting a User

Use the **Delete** action on any user row or from the edit page. You will be asked to confirm before the user is removed.

You can also select multiple users using the checkboxes and use the **Bulk Delete** action to remove them all at once.

> **Warning:** Deleting a user is permanent and cannot be undone.

## Impersonating a User

Administrators can temporarily log in as another user to troubleshoot issues or verify their experience. Click the **Impersonate** action on any user row to switch to that user's session.

While impersonating, a banner appears at the top of the page with an option to **stop impersonating** and return to your own account.

> **Note:** All actions performed while impersonating are logged under the impersonated user's account.
