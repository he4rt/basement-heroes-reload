# Teams

Teams let you organize users into logical groups for collaboration and access management. Navigate to **User Management > Teams** to view and manage teams.

## Team List

The teams table shows:

- **Name** — The team name with its slug shown below (searchable)
- **Status** — The current status of the team, displayed as a colored badge
- **Members** — A count of how many users belong to this team
- **Created / Updated / Deleted** — Timestamps for the team's lifecycle (some hidden by default; use column toggles to show them)

You can filter the list to show **active teams only**, **archived/deleted teams**, or **all teams** using the trashed filter.

## Creating a Team

Click **New Team** and fill in:

- **Owner** — Select the user who owns and manages this team (required)
- **Name** — The team's display name (required)
- **Description** — A brief description of the team's purpose (required)
- **Slug** — A URL-friendly identifier for the team (required)
- **Status** — The team's initial status (required)
- **Contact Email** — An email address for the team (required, must be a valid email)

## Editing a Team

Click **Edit** on any team row to modify its details. From the edit page, you can also manage team members.

## Managing Members

On the team edit page, the **Members** section lets you:

- **Add existing users** — Click **Attach** to search for and add existing users to the team
- **Create new members** — Click **Create** to add a brand new user directly as a team member
- **Remove members** — Use **Detach** to remove a user from the team (the user account itself is not deleted)
- **Edit member details** — Click **Edit** on any member row to update their information

## Deleting and Restoring Teams

Teams support **soft deletion**, meaning deleted teams are not permanently removed right away.

- **Delete** — Moves the team to the trash. It will no longer appear in the default list.
- **Restore** — Brings a soft-deleted team back to active status. Use the trashed filter to find deleted teams, then click **Restore**.
- **Force Delete** — Permanently removes a team. This action cannot be undone.

> **Tip:** Use the trashed filter dropdown above the table to view deleted teams and restore them if needed.
