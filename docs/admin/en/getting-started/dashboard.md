# Dashboard

The dashboard is the first page you see after logging in. It provides a quick overview of your account and the platform.

## Widgets

The dashboard displays two widgets:

- **Account Widget** — Shows your name and account information at a glance
- **Filament Info Widget** — Displays the platform version and helpful links

## Navigation

From the dashboard, use the sidebar to navigate to any section of the admin panel. You can always return to the dashboard by clicking the application logo or the **Dashboard** link at the top of the sidebar.
