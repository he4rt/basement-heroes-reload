# Navigating the Panel

The admin panel is organized into sections accessible from the sidebar on the left side of the screen.

## Sidebar

The sidebar is your main navigation tool. It contains links to all sections of the platform, organized into groups. You can collapse the sidebar by clicking the toggle button at the top to give yourself more screen space.

## Navigation Groups

Resources are organized into the following groups:

- **User Management** — Users, Teams, and Roles
- **System** — Activity Logs and Email Management
- **Documentation** — Opens the built-in knowledge base (this guide)

## Global Search

The search bar at the top of the page lets you quickly find records across the entire platform. Start typing a name, email, or keyword and matching results will appear instantly. Search works across users, roles, and documentation pages.

## Contextual Help

On every resource page (Users, Teams, Roles), look for the **help icon** in the sidebar footer. Clicking it opens a popover with links to documentation relevant to the page you're on. You can preview docs in a slide-over panel without leaving your current page.

## Documentation Mode

When you open the Knowledge Base from the sidebar, the navigation switches to a documentation tree showing all available guides. A **Back** button at the top lets you return to the regular admin panel navigation.
