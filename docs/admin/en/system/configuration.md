# Configuration

Key configuration options available in the Sycorax platform.

## Theme

The panel supports both **light** and **dark** mode. The theme mode is applied automatically based on your system preferences. Individual users can also customize their **primary accent color** from the Profile page — this is a personal setting and does not affect other users.

## Localization

The platform supports multiple languages:

- **English** (en)
- **Portuguese - Brazil** (pt_BR)

Users can switch their preferred language from the **Profile** page. All labels, navigation groups, system messages, and documentation will update to the chosen language.

## Documentation Cache

The built-in knowledge base caches documentation pages for faster loading. By default, pages are cached for **24 hours**. If you notice that documentation changes are not appearing, contact your system administrator to clear the cache.
