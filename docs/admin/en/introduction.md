# Welcome to Sycorax

Sycorax is your central management platform for users, teams, roles, and system administration. This knowledge base is your go-to reference for everything you can do in the admin panel.

## How This Guide Is Organized

- **Getting Started** — Learn how to navigate the panel, understand the dashboard, and customize your profile
- **User Management** — Create and manage users, assign roles and permissions, and organize teams
- **System** — Monitor activity logs, review sent emails, and adjust platform settings

## Need Help?

Look for the **help icon** in the sidebar footer on any page. It will show you documentation related to the page you're currently viewing. You can also use the **global search bar** at the top to find any topic quickly.
